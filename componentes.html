Componentes o elementos de Tailwind

a) Buttons

➤ ¿Para qué se usan? Los botones permiten ejecutar acciones en una página (enviar formularios, abrir menús, confirmar, etc.). En Tailwind se construyen con clases utilitarias para definir colores, bordes y efectos visuales. 

<!-- Clases principales:
px-4 py-2 → tamaño interno (padding)
rounded → bordes redondeados
bg-* y text-* → colores
hover:bg-* → color al pasar el cursor
cursor-not-allowed → desactiva interacción -->

<button class="px-4 py-2 rounded bg-blue-600 text-white hover:bg-blue-700">Primario</button>
<button class="px-4 py-2 rounded border hover:bg-gray-100">Secundario</button>
<button class="px-4 py-2 rounded bg-red-600 text-white hover:bg-red-700">Peligro</button>

b) Lista de elementos (cómo organizar una lista)

➤ ¿Para qué se usan? Las listas sirven para mostrar información ordenada o en grupo (pasos, tareas, menús, etc.). Tailwind permite personalizarlas con divide-y (líneas divisorias), border (bordes) y tipos de lista (list-disc o list-decimal).

  • Tarea 2: Diseñar la interfaz
  • Tarea 3: Pruebas y entrega

  • Responsivo

  • Fácil de usar
<!-- divide-y: crea líneas entre ítems
list-disc / list-decimal: viñetas o números
p-* y pl-* : padding (espaciado interno) -->

<ul class="divide-y divide-gray-200 border rounded">
  <li class="p-3">Elemento 1</li>
  <li class="p-3">Elemento 2</li>
</ul>

c) Navbar

➤ ¿Para qué se usa? La barra de navegación (Navbar) permite moverse entre secciones de una página. Con Tailwind se logra fácilmente usando flex para alinear los elementos y details para hacerla responsiva sin JavaScript.

Tailwind
Menú
Inicio Productos Contacto
  • Inicio
  • Productos
  • Contacto
<!-- flex: organiza elementos en fila
justify-between: separa extremos (logo y menú)
md:hidden / md:flex: visibilidad según tamaño de pantalla -->

<nav class="border rounded p-3">
  <div class="flex items-center justify-between">
    <span class="font-bold text-lg">MiMarca</span>
    <details class="md:hidden">
      <summary>Menú</summary>
      <a href="#">Inicio</a>
    </details>
  </div>
</nav>

d) Dropdown

➤ ¿Para qué se usa? El dropdown (menú desplegable) muestra opciones adicionales al usuario sin ocupar mucho espacio. En Tailwind se logra con <details> + <summary>, usando absolute para que flote y shadow para el efecto visual.

Opciones
  • Perfil
  • Configuración
  • Salir
<!-- relative: permite ubicar elementos flotantes
absolute: posiciona el menú sobre el contenido
shadow: agrega sombra al panel desplegable -->

<details class="relative inline-block">
  <summary class="px-4 py-2 rounded border bg-white hover:bg-gray-100">Opciones</summary>
  <ul class="absolute mt-1 rounded border bg-white p-1 shadow">
    <li><a href="#">Perfil</a></li>
  </ul>
</details>

e) Formularios

➤ ¿Para qué se usan? Los formularios permiten capturar datos del usuario (nombre, correo, selección, etc.). Tailwind usa clases utilitarias para aplicar bordes, colores y efectos de foco de forma rápida y uniforme.

Preferencias 
<!-- border: define el contorno del campo
rounded: bordes suaves
focus:ring: resalta el campo activo con color
grid / md:grid-cols-2: organiza en columnas responsivas -->

<form class="border rounded p-6 max-w-md">
  <input type="text" class="border rounded px-3 py-2 focus:ring focus:ring-blue-300">
  <select class="border rounded px-3 py-2">
    <option>Básico</option>
  </select>
</form>
